Sistema Inteligente de Rutas de Transporte

Este proyecto implementa un sistema inteligente basado en conocimiento
utilizando reglas lógicas y búsqueda heurística.

El sistema permite encontrar la mejor ruta entre dos estaciones de
un sistema de transporte utilizando el algoritmo de búsqueda BFS.

Tecnologías utilizadas:
- Python

Ejecución del sistema:

1. Ejecutar el archivo principal:

python rutas.py

2. Ingresar estación origen y destino.

3. El sistema mostrará la mejor ruta encontrada.
